# github-final-project
## Simple Interest Calculator

### Input:
- **p**: Principal amount
- **t**: Time period in years
- **r**: Annual rate of interest

### Output:
- **Simple Interest**: \( p \times t \times r \)

---
Example Usage:

```plaintext
Input:
   p = 1000
   t = 2
   r = 0.05

Output:
   Simple Interest = 1000 * 2 * 0.05 = 100
